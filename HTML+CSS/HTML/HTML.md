## 10. 字符编码

编码：将字符转换为二进制码

解码：将二进制码转换为字符

字符集（`charset`）：编码和解码所采用的规则。

乱码问题：编码与解码所采用的字符集不同。

常见字符集：

1. ASCII
2. ISO88591
3. GB2312
4. GBK
5. UTF-8(万国码，就用它)

可以通过meta标签设置网页的字符集：`<meta charset = "utf-8">`

## 14. 实体

在网页中编写的多个空格默认情况会自动被浏览器解析为一个空格。

在HTML中有些时候，我们不能直接书写一些特殊符号。

​	比如多个连续的空格，字母两侧的大于号、小于号。

解决方法：实体（转义字符）

​	空格：`&nbsp;`

​	大于号：`&gt;`

​	小于号：`&lt;`

​	版权符号：`&copy;`

## 16. 语义化标签

关注标签语义，而非标签样式。在页面中独占一行的元素称为块元素(block element)，不会独占一行的为行内元素。

标题标签：一般一个页面只有一个h1，基本只会用到h1~h3。块元素。

`<p>`：表示一个段落。块元素。

`<hgroup>`：为标题与段落分组。

`<em>`：表示强调，斜体。行内元素。

`<strong>`：强调，加粗。

`<blockquote>`：较长的引用。缩进换行。块元素。

`<q>`：较短的引用。加上引号。行内元素。

`<br>`：换行。

## 17. 块与行内

### 块元素（block element）

在网页中一般通过块元素对页面进行布局。

### 行内元素（inline element）

主要用来包裹文字。

一般在块元素中放行内元素。块元素里基本都能放。

p元素中不能放任何块元素。

浏览器解析网页时，会自动对网页中不符合规范的内容进行修正。

​	如：标签写在根元素外部、p中嵌套了块元素等......

## 20. 超链接

超链接可以从一个页面跳转到其他页面，或者是当前页面的其他位置。行内元素，可嵌套除它自身的任何元素。

href属性 指定跳转的目标路径（外部网站或内部其他位置）

## 21. 相对路径

跳转到服务器内部其他页面时，会使用相对路径。

> ./：当前文件所在目录（可省略）
>
> ../：表示当前文件目录的上一级
>
> ../../：上上级目录

## 22. 超链接其他用法

属性target 指定超链接打开位置

可选值：

1. _self：当前页面打开
2. _blank：新页面

回到页面顶部：href=“#”

跳转页面其他位置：href=“#id”（id，唯一不重复，元素的唯一标识）

在开发中，可以将#作为超链接的路径的占位符使用。

可以使用`javascript:;`来作为href的属性，此时点击超链接什么也不会发生。

## 23. 图片标签

用于引入外部图片。自结束标签。本质为替换元素（块和行内元素之间，具有两者特点）。

`<img alt="" src="">`

src：外部图片路径

alt：图片描述。默认情况下不显示，不加载时会显示，搜索引擎搜索的依据。

width：图片宽度。

height：图片高度。宽、高变化了其中一个，另一个会等比例变化。



一般情况在pc端，不建议修改图片的大小，需要多大就裁多大。

## 24. 图片格式

图片格式

* jpeg(jpg)：支持颜色比较丰富，不支持透明，不支持动图。

  照片

* gif：支持颜色较少，支持简单透明，支持动图

  颜色单一的图片、动图

* png：支持颜色丰富，支持复杂透明，不支持动图。

  颜色丰富，复杂透明图片（适用于网页）

* webg：谷歌新推出的，专门用于表示网页中图片的一种格式，具备其他图片格式的所有优点，文件比较小。

  兼容性不好。

* base64：将图片使用base64编码，将图片转换为字符。

  需要和网页一起加载的图片。

效果一样，用小的；效果不一样，用效果好的。

## 26. 音视频播放

`<audio>`：音频，替换元素。`<audio src="">`

音视频文件引入时，默认情况下不允许用户自己控制。添加属性controls（允许用户控制播放）、autoplay（自动播放）。

但目前大部分浏览器不会对音乐自动播放。

属性loop：是否循环播放。

还可以通过source属性指定文件。

```html
<audio controls>
    <!--不显示音频时，会显示文字-->
    对不起，您的浏览器不支持播放音频
    <!--第一个音频不支持时，会使用第二个音频-->
	<source src="......">
    <source src="......">
</audio>
```

```html
<!--可支持ie等浏览器，自动播放-->
<embed src="......" type="audio/mp3" width="400" height="100">
```

`<video src="">`：视频，使用方式与音频基本一样。

`<iframe>`引入网络音视频。
